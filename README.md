alphanum sorter
===============

Alphanum sorter is based on Dave Koelle algorithm.

This npm module is based on original work published at http://www.davekoelle.com/files/alphanum.js

Installing
----------
```
    npm i alphanum
```

How to use?
-----------

```javascript
    require("alphanum");


    my_list=['1.10', '1.1', '1.2'];
    my.list.sort(key=alphanum);
```

Development
-----------

Running tests:
```
    npm test
```
